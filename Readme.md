# DIA TAJ - Premium Diamond Import & Export 

**Welcome to my diamond showcase website!** This platform is designed to provide a seamless experience for exploring and learning about premium diamonds, representing **DIA TAJ - Excellence in Diamond Trading Since 1996.**

## Features

- **Interactive Diamond Showcase:** Explore various diamond categories including Rough Work, Polish Work, CVD Diamonds, and Jewellery Export.
- **Detailed Specifications:** In-depth information regarding diamond imports and exports.
- **Modern User Interface:** A premium, user-friendly UI utilizing modern CSS animations, gradients, and interactive components.
- **Educational Resources:** Dedicated sections to learn about the diamond industry and processes.
- **Fully Responsive:** Optimized design that works perfectly on all devices, from desktops to mobile phones.

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript  
- **Libraries & Frameworks:** Bootstrap, Animate.css, Swiper, GLightbox, and Leaflet Maps (for location pinning).  
- **Fonts & Icons:** Google Fonts (Inter, Playfair Display) and Bootstrap Icons.

## Execution Steps (How to Run the Project Locally)

**Since this project is a static website**, it does not require a complex backend configuration.

### Method 1: Direct File Execution (Simplest)

1. Clone or download the repository  
2. Extract the folder if zipped  
3. Navigate to project folder  
4. Open `index.html`  

### Method 2: Using VS Code Live Server

1. Install **Visual Studio Code**  
2. Open project folder  
3. Install **Live Server** extension  
4. Open `index.html`  
5. Click **"Go Live"**  

### Method 3: Using Python

1. Open terminal  
2. Navigate to project folder  
3. Run: 
```bash
python -m http.server 8000
```

4. Open browser → **http://localhost:8000**
